# Introduction to Smart Contracts

Introduction to Ethereum Smart Contracts using the official
[Solidity guide](https://solidity.readthedocs.io/en/latest/introduction-to-smart-contracts.html).

## Project setup 

Execute the `testrpc` daemon using `docker-compose`:

```bash
$ docker-compose up -d
```

You can test the contracts using the `truffle test` command:

```bash
$ docker exec eth truffle test
```
